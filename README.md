# TP Django - MonProjet

## Description
Projet Django avec 3 pages, héritage de templates et fichiers statiques.

## Fonctionnalités
- Pages : Accueil, À propos, Contact
- Héritage de templates (base.html)
- Navigation cohérente
- CSS personnalisé
- Configuration Django complète

## Installation
\`\`\`bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python manage.py runserver
\`\`\`

## Structure
- MonProjet/ : Configuration Django
- monapp/ : Application principale
- monapp/templates/ : Templates HTML
- monapp/static/ : CSS et images

## Technologies
- Django 6.0.2
- HTML5/CSS3
- SQLite
- Python 3
